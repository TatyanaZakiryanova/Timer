## Technologies Used

- Typescript
- React
- Jest
- React Testing Library
- Material UI

## How to start project

in the project directory enter:

```js
npm install
```

and then run:

```js
npm start
```

build the project:

```js
npm run build
```

tests:

```js
npm test
```
